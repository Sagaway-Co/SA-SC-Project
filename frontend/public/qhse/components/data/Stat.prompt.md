Animated count-up KPI metric — counts from 0 on scroll-into-view. The site's "trust number".

```jsx
<Stat value={1240000} suffix="+" label="隐患已闭环" sublabel="Hazards resolved" />
<Stat value={96.4} decimals={1} suffix="%" label="平均闭环率" align="center" />
```

Props: `value`, `prefix`, `suffix`, `decimals`, `label`, `sublabel`, `duration` (ms), `align`.
