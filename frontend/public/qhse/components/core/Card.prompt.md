Layered dark surface container for feature grids, panels, and dashboard tiles.

```jsx
<Card interactive spotlight padding={28}>
  <Eyebrow>证据链</Eyebrow>
  <h3>不可篡改</h3>
  <p>北斗定位 + SHA-256 指纹</p>
</Card>
```

Props: `padding`, `interactive` (hover lift + accent border), `glow` (ambient halo), `spotlight` (cursor-follow radial highlight).
