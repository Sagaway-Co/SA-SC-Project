Dark-UI text field with stacked label, lime focus ring, hint/error text, and optional leading icon.

```jsx
<Input label="企业邮箱" placeholder="you@company.com" iconLeft={<MailIcon />} />
<Input label="手机号" error="请输入有效号码" />
```

Props: `label`, `hint`, `error`, `iconLeft`, `size` (`md | lg`), plus native input attrs.
